Describe and Cite the API

I used the openweathermap API, which allows you to query weather information for a specific geographic area and get a JSON document with requested info in return (https://openweathermap.org/).

API Request Options

I used two specific queries. First, I used "weather?q=Boston" to query the weather around Boston. Second, I used "units=metric" because I'm not a barbarian. The thrid query string is the API ID. The API call resovles to: http://api.openweathermap.org/data/2.5/weather?q=Boston&units=metric&APPID=#####

Potential Applications

This API would be useful whenever a web application would need to query weather data. It can be used to predict road safety conditions as well as be used as a data point to recommend what someone should wear on a certain day
